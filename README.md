To run the project on your device make sure you have the following software :

Python should be installed on the system (https://www.python.org/).
Install the Django framework using the following command: pip install Django
Install the request as it's important to fetch the data use the following command: pip install requests
Open the source code folder in a terminal and use the following command to run the server: python manage.py runserver This should run the project on the web server http://127.0.0.1:8000/users/signup Thank you <3
